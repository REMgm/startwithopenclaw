## What This Project Is
StartWithOpenClaw.com is a beginner-focused educational website and affiliate marketing platform for OpenClaw — an open-source self-hosted AI agent framework. The site provides free courses, setup guides, troubleshooting resources, and hosting provider comparisons.

## Site Structure

### Main Pages
- **Homepage** (`/`) — Landing page with value prop, hosting comparison, FAQ
- **Course** (`/course/`) — 10-module interactive beginner's course
- **Mac Mini Guide** (`/mac-mini/`) — Complete self-hosting setup guide
- **Bugfixing** (`/bugfixing/`) — Troubleshooting guide for common errors
- **Blog** (`/blog/`) — SEO content marketing (articles published regularly)
